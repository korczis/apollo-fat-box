# Apollon

Your own personal cluster built on top of the cloud.

## Prerequisites

- CoreOS
- Docker
- Etcd

## Technologies

- Docker (engaged)
- Fleet (engaged)
- Flannel
- Kubernetes
- Mesos
- AWS Route 53

## Cloud providers

- Digital Ocean
- AWS
